# Usage

## Collector
### Environment variables
* FLUFFY_LOG_LEVEL - set to one of debug, info, warn, error. Default is info
* FLUFFY_TURSO_URL - required, libsql url for database
* FLUFFY_TURSO_AUTH_TOKEN - required, token for libsql url
* FLUFFY_GATE_BUCKET_SIZE - default 25, higher number runs risk of more deny, lower number means collection is slower
